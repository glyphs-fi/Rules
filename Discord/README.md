# Discord Rules

In this folder, we have some files that [GlyfiBot](https://github.com/glyphs-fi/GlyfiBot) posts in our [`#📜｜rules-n-info`](https://discord.com/channels/850457945220841473/850470869955772436) channel.

Each file corresponds to a single message.

The numbering of the files ensures that the messages will be sent in the correct order.

You can put any of the following file types in here, and the bot will send it:
- Media files (`.png` `.jpg` `.jpeg` `.gif` `.webp` `.webm` `.mp4`)
- Text files (`.md` `.txt`)
  - These get relayed straight through to Discord, no processing, so keep in mind that these files are in _Discord Markdown_ format, **not** _GitHub Markdown_ format!  
  So watch your newlines!
- Discord Message JSON files (`.json`)
  - These can be generated with https://discordutils.com/embed-builder
