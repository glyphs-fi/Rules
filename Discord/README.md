# Discord Rules

In this folder, we have some files that [GlyfiBot](https://github.com/glyphs-fi/GlyfiBot) posts in our `#📜｜rules-n-info` channel.

Each file corresponds to a single message.

The numbering of the files ensures that the messages will be sent in the correct order.

You can put any of the following files in here, and the bot will send it:
- Media files (".png" or ".jpg" or ".jpeg" or ".gif" or ".webp" or ".webm" or ".mp4")
- Text files (".md" or ".txt")
  - These get relayed straight through to Discord, so keep in mind that these files are in _Discord Markdown_ format, **not** _GitHub Markdown_ format!
- Discord Message JSON files (".json")
  - These can be generated with https://discordutils.com/embed-builder
